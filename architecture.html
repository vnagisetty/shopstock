Data Flows

🔄Login sync → Sheets → IndexedDB

Issues
⚡ No Sheets transaction isolation
⚡ Row-scan on every update O(n)
⚠ Custom multipart parser
⚠ Offline create: no item_id
⚠ Queue stops on first error
⚠ No optimistic UI updates
⚠ Sync errors not shown in UI
ℹ Session expires in 7d (no refresh)
ℹ Drive links permanent (no cleanup)
ℹ No rate limiting on API routes
ℹ Icon dimensions hardcoded 200px
ℹ No batch write endpoints
